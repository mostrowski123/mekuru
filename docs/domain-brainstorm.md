# Domain Name Brainstorm for Mekuru

## Context

Mekuru (めくる — "to turn pages") is a Japanese-first EPUB and manga reader with vertical text rendering, OCR, dictionary lookups, MeCab analysis, and vocabulary management. We're looking for a memorable, SEO-friendly domain that's actually available to register.

## Important: "mekuru" Brand Conflicts

My research found several existing sites already using the "mekuru" name:

| Domain | Owner |
|--------|-------|
| **mekuru.co** | めくる! — Japanese event info site |
| **mekuru.site** | MEKURU — manga trial-reading service (1,100+ works) |
| **mekuruapp.com** | Mekuru App — JLPT-focused Japanese learning app |
| **atelier-mekuru.com** | Atelier MEKURU |
| **mekuru.id** | Mekuru Ramen House (Indonesia) |

This means **mekuru.com** is very likely taken or parked, and the brand has competition in the Japanese learning/reading space specifically.

## Domain Ideas

### Tier 1: Mekuru Brand Variations (if you want to keep the name)

| Domain | Why it could work | SEO notes |
|--------|-------------------|-----------|
| `getmekuru.com` | Common SaaS pattern ("get + brand") | Brand-first, less organic SEO |
| `mekuru.dev` | `.dev` signals a tech product | Lower SEO weight than `.com` |
| `mekuru.read` | `.read` TLD is very on-brand | Newer TLD, may not be available |
| `usemekuru.com` | Another common SaaS pattern | Brand-first |
| `mekuru.tools` | Signals a utility app | Neutral SEO |
| `mekureader.com` | Combines brand + function | Good for "reader" keyword searches |
| `mekurureader.com` | Full brand + "reader" — no ambiguity | Strong for "reader" keyword searches; 14 chars is longer but very clear |

### Tier 2: Japanese-Themed Alternatives

| Domain | Meaning | Why it could work |
|--------|---------|-------------------|
| `yomimekuru.com` | 読みめくる (read + turn pages) | Very Japanese, descriptive |
| `tategaki.app` | 縦書き (vertical writing) | Unique differentiator, strong SEO for vertical text |
| `dokusho.app` | 読書 (reading) | Clean, common Japanese word |
| `yomitai.app` | 読みたい (want to read) | Expressive, memorable |
| `tsundoku.app` | 積ん読 (buying books and not reading them) | Fun, well-known Japanese word internationally |

### Tier 3: SEO-Optimized Descriptive Domains

| Domain | Target keywords |
|--------|-----------------|
| `japanesereader.app` | "japanese reader" |
| `jpepub.com` | "jp epub" |
| `mangadoku.com` | "manga" + 読 (read) |
| `nihonreader.com` | "nihon reader" / "japanese reader" |
| `epubyomi.com` | "epub" + 読み (reading) |

## Recommendations

1. **If you want maximum SEO**: Go with a descriptive `.app` domain like `tategaki.app` or `dokusho.app` — these match search intent for people looking for Japanese reading tools.

2. **If you want to keep the Mekuru brand**: Try `mekurureader.com`, `getmekuru.com`, or `mekureader.com` — but be aware of brand confusion with the existing `mekuruapp.com` (JLPT learning app). `mekurureader.com` appears to have no existing presence online, making it a strong candidate.

3. **If you want something unique and memorable**: `tsundoku.app` is a widely-known Japanese word in English-speaking communities and would stand out.
