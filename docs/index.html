Vocabulary & Anki

Bookmarks & Highlights

Reader Customization

Themes, color modes, adjustable margins and font size, swipe sensitivity, and per-book quick settings.

Manga OCR Pro

Cloud OCR for CBZ manga pages with dictionary lookup. Auto-crop and custom self-hosted server support.

Mekuru is a Japanese-first EPUB and manga reader designed for language learners and native readers alike. Read vertically, look up words offline, build vocabulary with Anki export, and run OCR on manga — all in one app.
