# Managing Dictionaries

You can open the **Dictionary Manager** from the top-right button on the **Dictionary** tab or from **Settings > Manage Dictionaries**.

## What You Can Do

- **Import** - tap **+** to import a Yomitan `.zip` file or a Yomitan collection `.json` backup
- **Reorder** - drag the handle on the left to change priority
- **Enable or disable** - use the switch on each row
- **Delete** - use the delete action for a permanent removal

## Priority Order

Dictionary order controls how grouped lookup results are sorted. Higher-priority dictionaries appear first when multiple dictionaries match the same word.

## Empty State

If you have not imported any dictionaries yet, the screen shows a **Browse Downloads** shortcut so you can jump directly to the built-in Downloads screen.

## Notes

- Disabled dictionaries stay installed but are excluded from search and tap-to-lookup.
- Deleting a dictionary removes all of its imported entries, so you will need to re-import it later if you want it back.
