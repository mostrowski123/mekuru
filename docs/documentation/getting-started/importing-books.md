# Importing Books (EPUB)

Mekuru supports EPUB files for text-based books and novels.

## How to Import

1. Open the **Library** tab.
2. Tap the **+** button.
3. Choose **EPUB**.
4. Select an `.epub` file from the system file picker.
5. The book is imported into your library grid.

## What Gets Imported

When you import an EPUB, Mekuru extracts:

- **Title** from the book metadata
- **Cover image** when one is present
- **Language** and writing-mode hints used for reader defaults
- **Table of contents** data for chapter navigation

## File Size Limit

EPUB files up to **200 MB** are supported.

## Managing EPUB Books in the Library

- **Open a book** - tap its cover
- **Open book actions** - long-press a library item
- **Sort the library** - choose **Date imported**, **Recently read**, or **Alphabetical**
- **Rename** - available from the long-press actions
- **Change Cover** - available from the long-press actions
- **Delete Book** - available from the long-press actions
- **Bookmarks and Highlights** - EPUB entries expose these from the long-press actions

## Reading Features

EPUB books support the full reader feature set:

- Vertical and horizontal text rendering
- Tap-to-lookup dictionary integration
- Bookmarks and highlights with notes
- Quick per-book reader settings
- Reading progress auto-save and resume

For image-based manga, see [Importing Manga](getting-started/importing-manga.md).
