# Downloads

Mekuru includes a built-in **Downloads** screen under **Settings > Downloads**. These free packs can be installed and removed without leaving the app.

## Built-in Dictionaries

| Download | What it adds |
|-|-|
| **JMdict English** | Standard Japanese-English dictionary |
| **JMdict English with Examples** | JMdict plus example sentences |
| **KANJIDIC** | Kanji dictionary data |

These downloads are imported automatically and appear in the Dictionary Manager like any other installed dictionary.

## Lookup Enhancements

| Download | What it adds |
|-|-|
| **KanjiVG** | Stroke order diagrams in compatible dictionary results and single-kanji searches |
| **JPDB Frequency** | Word frequency ranks shown in dictionary results |

## Sources

- **JMdict / KANJIDIC** - from the EDRDG projects, distributed through Yomitan releases
- **KanjiVG** - from the [KanjiVG project](https://kanjivg.tagaini.net/)
- **JPDB Frequency** - based on JPDB data distributed through yomitan-dictionaries

## Removing Downloads

The same **Downloads** screen also lets you delete installed packs later if you want to free storage or replace them with another source.
